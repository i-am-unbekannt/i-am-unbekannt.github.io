Installation On Linux

● Update your system sudo apt-get update -y.

● Install requirements for the Installation progress sudo apt-get install git python3 python3-pip mdk3 macchanger.

● Clone the repository git clone https://github.com/i-am-unbekannt/unbekannt-framework.

● cd in to the folder cd unbekannt-framework.

● Install python requirements pip3 install pathlib tcping tercolor webbrowser colorama.

● Run the Unbekannt Framework sudo python3 Unbekannt.py.

INFO

For more help inside the Unbekannt Framework, use the 'help' command.
