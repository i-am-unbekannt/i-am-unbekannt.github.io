Introduction #

The Unbekannt Framework is a Hacking and Pentesting tool for Windows.

There is also a Linux version 6.0, but this version is no longer supported.

The Unbekannt Framework allows to run multiply modules for network attacking, network sniffing, DDoS attack, reverse TCP shell and more. There are also tools/module for IP lookup/ping and port scanning. Over the time this tool gets more updates with more functions and modules.
